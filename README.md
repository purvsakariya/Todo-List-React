# 📝 React Todo App

A clean and modern Todo List application built using **React JS**, **Vite**, and **Tailwind CSS**.  
This project is inspired by the *Chai aur React* playlist and focuses on building strong fundamentals in React.

---

## 🚀 Features

- ✅ Add new todos
- ✏️ Edit existing todos
- ❌ Delete todos
- ✔️ Mark todos as completed
- 💾 Task Save in Localstorage
- 🎨 Clean UI using Tailwind CSS

---

## 🛠️ Tech Stack

- **React JS** (Functional Components & Hooks)
- **Vite** (Fast build tool)
- **Tailwind CSS** (Utility-first styling)
